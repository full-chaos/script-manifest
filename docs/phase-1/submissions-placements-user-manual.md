# Submissions + Placements User Manual

## What this adds

Submissions now include placement recording and verification controls in `/submissions`.

## How to use it

1. Record or load submissions in `/submissions`.
2. Click `Record placement`.
3. Choose the target submission and placement status.
4. Create the placement.
5. Use `Mark verified` or `Mark rejected` to update verification state.

## What is shown

For each submission, the page shows:

- placement ID
- placement status
- verification state
- action buttons for verification updates
