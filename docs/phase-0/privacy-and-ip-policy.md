# Privacy and IP Protection Policy (v0)

## Writer IP Rights

- Writers retain all rights to uploaded scripts and related materials.
- Platform receives only limited license to host, display, and process content per user settings.

## Access Control Guarantees

- Private projects are inaccessible without explicit grant.
- Discoverable projects honor writer-configured access level and download permissions.
- Every script file access is logged and visible to the writer when applicable.

## Data Protection

- Encrypt data in transit (TLS) and at rest where supported.
- Apply least-privilege access to all internal services.
- Maintain auditable history for permission changes and downloads.

## Third-Party Sharing

- No sale of script content.
- Limited sharing only for operational processors (storage, email, payments), under DPAs.
