# Content Moderation and Trust Policy (v0)

## Scope

Applies to scripts, feedback notes, profile text, and direct messages.

## Prohibited Conduct

- Harassment, hate speech, and threats
- Plagiarism or unauthorized script uploads
- Scam behavior and impersonation
- Spam, token farming, and collusive manipulation

## AI-Generated Content Rules

- AI-assisted writing is allowed only with user disclosure if requested by a partner/program.
- Fully generated script submissions may be restricted by competition-specific rules.
- Synthetic feedback spam is prohibited.

## DMCA Workflow

1. Intake complaint through designated email/form.
2. Triage and temporary restriction where warranted.
3. Notify uploader and allow counter-notice process.
4. Log all actions in moderation audit history.

## Enforcement Ladder

- Warning
- Content removal
- Temporary suspension
- Permanent account ban

All enforcement actions must be traceable in an immutable moderation audit trail.
